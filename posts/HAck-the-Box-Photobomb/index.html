Hack The Box - Photobomb

Posted Feb 12, 2023 Updated Feb 12, 2023
By Niraj Maharjan
4 min read

Photobomb was a easy rated linux box, where plaintext credentials was discovered after viewing the source code. These credentials then lead to a webpage with downloadÂ  functionality that was vulnerable to blind command injection. User was able to run the script with sudo permission and path hijacking was used to escalate privileges to root.

COMMAND INJECTION

while testing for command injection, when we add sleep 5 on photo or dimension parameter we get 500 server error response.

But when we add sleep 5 on filetype parameter we get response after 5400 milisecs, this confirms that thereâs a command injection vulnerability.
